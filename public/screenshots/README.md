# Captures d'écran de RecrutIA

Dépose ici tes vraies captures (format PNG, idéalement ~2400×1500 px, en clair).
Les noms de fichiers doivent être EXACTEMENT :

- dashboard.png          → le tableau de bord
- analyse-candidat.png   → une fiche candidat analysée (score + forces/faiblesses)
- pipeline.png           → le pipeline de recrutement
- comparateur.png        → le comparateur de deux candidats

Dès qu'un fichier est présent, il remplace automatiquement le cadre gris
sur la page /recrutia. Pas besoin de toucher au code.

Astuce : charge la démo dans RecrutIA (bouton "Charger une démo" du dashboard)
pour avoir de belles données à capturer.
